Javascript n'est pas un langage orienté objet comme les autres! La preuve:

JavaScript uses prototypes and don't have classes (and methods) like many object oriented languages. A JavaScript developer need to think in JavaScript.

voir StackOverflow

Voici un exemple de définition de classe:

Session = function () {
	this.root = DEFAULT_ROOT;
	this.page = DEFAULT_PAGE;
	this.footer = 'footer';
	this.content = 'content';

	this.private_getPage = function(url) {
		return $.urlParam('page', url, DEFAULT_PAGE);
	}

	this.private_getRoot = function(url) {
		return $.urlParam('root', url, DEFAULT_ROOT);
	}

	...

	this.load = function() {
		...
	}

	this.init = function() {
		...
	}
};

Et voici un exemple d'utilisation:

var session = new Session();
session.init();
session.load();
