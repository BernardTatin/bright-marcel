Voici quelques notes à propose de Git depuis la ligne de commande.

les tags

La liste des tags s'obtient facilement avec :

git tag

L'ajout d'un tag se fait tout aussi simplement:

git tag -a poems-1.0.1 -m 'blah bla'

Et pour l'envoi des tags dans le dépôt :

git push origin --tags

Pour créer une branche version2 depuis le tag v2.0.0:

git checkout -b version2 v2.0.0

log

La commande git log possède une option très intéressante : --decorate=full qui permet de visualiser un maximum de renseignents sur chaque commit, dont le tag correspondant.
